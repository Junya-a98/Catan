        グランドキャンペーン（全部入り）

        37タイルの探索、予告イベント、常設市場、公開競売、資源信用を同時に使います。港湾封鎖は予告時点で公開済みの交換所だけを対象とし、公開済みの交換所がなければその回は発動しません。

        手番の流れ

        1. ダイスを振る
        2. 建設・交易・発展カード
        3. 手番終了

        盗賊と「7」

        手札8枚以上なら半分（端数切捨て）を捨て、盗賊を別タイルへ動かします。隣接する相手からランダムに1枚奪えます。

        国内交易

        手番プレイヤーと他プレイヤーだけが交換できます。双方が1枚以上を渡し、同じ資源を両側には指定できません。ORにした欄は、承諾側が候補から実行する1種類を選びます。

        銀行・港

        通常は同じ資源4枚を任意の1枚へ交換。開拓地・都市が接続する港に応じて3:1または指定資源2:1になります。

        予告イベント

        全員へ影響する次回イベントと対象を事前公開します。発動までの残り手番と現在有効な効果を確認し、建設・交易・生産への影響を見越して行動してください。

        37タイル・フロンティア探索

        開始時は中央7タイルだけが見えています。霧に接する街道を建設すると未知タイルと港が公開され、未公開の資源・数字・港は生産や交易に使えません。

        常設市場と公開競売

        常設市場では、渡す資源と希望する資源を指定して注文を出します。公開競売では、出品者が品物と最低入札枚数を示し、ほかのプレイヤーは好きな資源の組み合わせで入札できます。出品・入札中の資源は取り置かれ、取消・期限切れでは手札へ戻ります。

        資源信用

        銀行から指定資源1枚を借り、期限までに同じ資源1枚と任意資源1枚をまとめて返済します。返済期限を過ぎると任意資源3枚の延滞債務へ変わり、分割返済できます。未返済中は公開VPが通常−1、延滞中は−2されます。

        CREATE MARKET ORDER

        常設市場へ注文を出す

        赤の「出品する」は取り置かれる資源、緑の「希望する」は購入者から受け取る資源です。

        出品と希望を1枚以上ずつ選んでください。

        同じ資源を出品側と希望側の両方には指定できません。希望は各資源19枚までです。

        CREATE AUCTION

        公開競売を開く

        出品する資源と、受け付ける最低入札枚数を指定します。

        出品する資源を1枚以上選んでください。

        出品資源と入札資源は、競売が終わるまで使用できない状態で取り置かれます。

        RESOURCE CREDIT

        資源を借りる

        受け取る資源を1種類選んでください。

        借りる資源を選んでください。

        借入中の債務とVP減点は、全プレイヤーへ公開されます。
